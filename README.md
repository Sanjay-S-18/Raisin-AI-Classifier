# 🍇 Raisin AI Classifier

A machine learning web application built with **Streamlit** that predicts whether a raisin belongs to the **Kecimen** or **Besni** class based on its physical characteristics.

---

## 🚀 Overview

This project uses a **Random Forest Classifier** trained on the Raisin Dataset.  
It takes numerical features like area, perimeter, axis lengths, and shape properties to classify raisins into two categories.

The app provides an interactive web interface where users can input values and get instant predictions with confidence scores.

---

## 🧠 Machine Learning Model

- Algorithm: Random Forest Classifier
- Criterion: Gini Index
- Number of Trees: 10
- Train/Test Split: 90% / 10%
- Random State: 42

---

## 📊 Dataset Features

The model is trained using the following features:

- Area  
- MajorAxisLength  
- MinorAxisLength  
- Eccentricity  
- ConvexArea  
- Extent  
- Perimeter
